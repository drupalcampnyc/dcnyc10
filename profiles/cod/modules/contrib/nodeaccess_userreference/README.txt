INSTALLATION
------------
1. Copy nodeaccess_userreference folder to modules directory.
   (Usually 'sites/all/modules')
2. At 'admin/build/modules' enable the Node access user reference module.


USAGE
-----
Create a User reference field in your content type, on the page where you 
configure the properties of your field, you will also see settings for granting
access to the referenced user.
